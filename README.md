# ED_Lista11.03
## Escalonamento Round Robin

Este desafio consiste em criar uma aplicação, em Java, que Simule os processos do Escalonamento Round Robin.

```
Requisitos do projeto:

1. Arquitetura MVC;
2. A fila deve ser importada de uma biblioteca própria(disponível em: https://github.com/Otavio-Olinto/ED_Lista09.03);
3. A fila deve decrementear a prioridade(qntRetornos) de cada processo a cada iteração;
4. Caso a qntRetornos continue maior que zero, o processo deve retornar a fila.
```

```
----------------------
Processos
----------------------
1. nome: String
2. qntRetornos: int
----------------------

Utilizar a seguinte massa de dados:

{"notepad.exe;14","write.exe;35","chrome.exe;27","acrobat.exe;52","firefox.exe;18","word.exe;25"}
```
